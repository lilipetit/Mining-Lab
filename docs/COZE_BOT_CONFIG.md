## 技能配置

### 1. 算法讲解技能

#### K-Means 聚类
```
核心概念：
- 无监督学习算法
- 将数据分成 K 个簇
- 目标：最小化簇内平方误差

关键参数：
- K 值：聚类数量，如何选择？
- 最大迭代次数：收敛控制
- 初始化方式：k-means++ vs random

评估指标：
- 惯性 (Inertia/SSE)：簇内距离之和
- 轮廓系数：簇内紧密度和簇间分离度
```

#### 决策树
```
核心概念：
- 有监督学习
- 递归分裂构建树
- 信息增益/基尼系数作为分裂准则

关键参数：
- 最大深度：控制树的复杂度
- 最小样本分割数：防止过拟合
- 剪枝策略：预剪枝 vs 后剪枝

评估指标：
- 准确率、精确率、召回率、F1
```

#### Apriori 关联规则
```
核心概念：
- 挖掘频繁项集
- 发现项之间的关联关系
- 经典应用：购物篮分析

关键指标：
- 支持度 (Support)：出现频率
- 置信度 (Confidence)：条件概率
- 提升度 (Lift)：实际提升程度

公式：
- Support(A→B) = P(A∩B)
- Confidence(A→B) = P(B|A)
- Lift(A→B) = Confidence(A→B) / P(B)
```

#### 其他算法
- **KNN**：基于距离的分类
- **朴素贝叶斯**：概率分类器
- **PCA**：主成分分析降维
- **回归**：线性/逻辑回归

#### 2. 经典数据集说明
```
📊 鸢尾花数据集 (Iris)
- 150 样本，4 特征，3 类别
- 特征：花萼/花瓣长度和宽度
- 用途：分类、聚类入门

📊 酒馆数据集 (Wine)
- 178 样本，13 特征，3 类别
- 来源：意大利葡萄酒化学分析
- 用途：多特征分类

📊 乳腺癌数据集
- 569 样本，30 特征，2 类别
- 特征：细胞核图像特征
- 用途：医学诊断

📊 购物篮数据
- 事务型数据
- 用于关联规则挖掘
```

#### 3. 常见错误与解决方案
```
❌ 问题：K-Means 聚类效果差
✅ 解决：
1. 检查数据标准化
2. 尝试不同的 K 值
3. 使用轮廓系数评估

❌ 问题：决策树过拟合
✅ 解决：
1. 限制最大深度
2. 增加最小样本数
3. 使用交叉验证

❌ 问题：Apriori 运行慢
✅ 解决：
1. 提高支持度阈值
2. 减少数据量
3. 使用 FP-Growth
```
